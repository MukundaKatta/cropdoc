# CropDoc

AI crop disease detector using computer vision for plant disease identification from leaf photos.

## About
CropDoc helps farmers and agronomists identify plant diseases early by analyzing photos of crop leaves. It uses computer vision models trained on plant pathology datasets to detect diseases, assess severity, and recommend treatment options - supporting healthier harvests and reduced crop loss.

## Tech Stack
- Python
- Computer vision (PyTorch, OpenCV)
- Click/Typer (CLI)
